# Enter School

Manajamen antrean untuk pendaftaran sekolah.

> Script SQL ada di folder `schemas/`<br>
> Jangan lupa buat file .env (salinan dari .env.example)

## Features

- Tanpa login, bisa langsung pakai app.
- Ambil Antrean (`Queue`) dengan nomor telepon (`Phone Number`).
- Lihat semua `Queue` yang telah diambil serta `Phone Number` nya.
- Lihat Detail `Queue` yang telah didaftar.
